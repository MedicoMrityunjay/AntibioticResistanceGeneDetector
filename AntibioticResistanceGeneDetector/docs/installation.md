# Installation

## Requirements
- Python 3.8+
- BLAST+ or DIAMOND (optional, for real mode)

## Install from PyPI
```sh
pip install arg-res-detector
```

## Install from TestPyPI
```sh
pip install --index-url https://test.pypi.org/simple/ --no-deps arg-res-detector
```

## Install from source
```sh
pip install .
```
